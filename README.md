# BazarFlow

## Technologies
* **Backend:** ASP.NET Core / .NET 9
* **Frontend:** Angular
* **Database:** SQL Server

## Features Implemented
* Login / Logout
* Permissions
* Cashier
* Products
* Batches
* Invoices
* Suspended invoices
* Adjustment requests
* Inventory
* Offers
* Reports
* Settings
* Exports
